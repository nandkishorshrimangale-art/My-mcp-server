Simple MCP Server

A Python-based Model Context Protocol (MCP) server that provides simple tools for AI assistants.

Features

- Add two numbers
- Multiply two numbers
- Calculate square of a number
- Greet users
- Easy to extend with new tools

Requirements

- Python 3.11 or higher
- MCP Python SDK

Installation

pip install -r requirements.txt

Running the Server

python server.py

Example Tools

Add Numbers

Input:

add(5, 3)

Output:

8

Multiply Numbers

Input:

multiply(4, 6)

Output:

24

Greet User

Input:

greet("Nandkishor")

Output:

Hello, Nandkishor! Welcome to the MCP Server.
